ADTClientDelegate Protocol Reference

Conforms to NSObject
Declared in ADTClientDelegate.h

Overview

The delegate of an ADTClient object must adopt the ADTClientDelegate protocol. These methods allow the delegate to receive ACR results and be notified upon errors.

When you register your application with AdTonik, you will be given an appID and an appSecret. Both of these must be specified in the delegate object.

Optional method to override the AdvertiserIdentifier method. AdTonik uses the IFA to pair the ad call to the tv data, therefore this method must return the same IFA used by your chosen ad sdk.

- (NSString *)ADTAdvertiserIdentifier

Discussion

By default, we use the IdentifierForAdvertising on iOS 6.0+ and the SHA1 hashed deviceIdentifier for previous iOS versions.
